# AI 祝福语生成器 - 微信小程序版本

这是一个将 Next.js 应用程序移植到微信小程序平台的版本。该应用程序可以根据用户提供的场景、人群和风格信息，利用 AI 生成个性化的祝福语。

## 项目结构

```
wechat-miniprogram/
├── app.js                 # 小程序入口文件
├── app.json               # 全局配置
├── app.wxss               # 全局样式
├── pages/                 # 页面目录
│   └── index/             # 首页
│       ├── index.js       # 页面逻辑
│       ├── index.wxml     # 页面结构
│       ├── index.wxss     # 页面样式
│       └── index.json     # 页面配置
├── utils/                 # 工具函数
│   ├── config.js          # 配置数据
│   ├── prompt-templates.js# AI提示词模板
│   ├── validation.js      # 输入验证
│   └── api-client.js      # API客户端
└── project.config.json    # 项目配置
```

## 核心功能

### 1. 双模式输入
- **智能描述模式**：用户可以通过自然语言描述需要祝福的场景
- **经典模板模式**：通过选择预设的场景、节日、人群和风格来生成祝福语

### 2. 丰富的场景配置
- 传统节日：春节、元宵节、清明节、端午节、七夕节、中元节、中秋节、重阳节
- 现代节日：元旦、情人节、白色情人节、妇女节、劳动节、母亲节、儿童节、父亲节、教师节、国庆节、圣诞节
- 人生时刻：生日、婚礼、毕业、生子、周年纪念
- 成就庆祝：升职、开业、获奖、考试成功
- 生活祝福：乔迁、康复、退休、日常问候

### 3. 多样化的目标人群
支持25种预设关系：朋友、家人、父母、兄弟姐妹、祖父母、同事、领导、老师、学生、同学、恋人、夫妻、子女、客户、长辈、晚辈等

### 4. 多种祝福风格
- 温馨亲切
- 正式庄重
- 幽默风趣
- 简洁明了
- 诗意优美
- 传统经典
- 现代时尚
- 感恩感谢
- 激励鼓舞

## 开发指南

### 环境要求
- 微信开发者工具
- 微信小程序账号

### 配置说明

1. **API域名配置**：在 `utils/api-client.js` 中修改API地址为你自己的后端服务地址
2. **小程序appid**：在 `project.config.json` 中修改为你的小程序appid

### 部署步骤

1. 使用微信开发者工具打开项目目录
2. 修改配置文件中的API地址和appid
3. 点击"上传"按钮上传代码
4. 在微信公众平台提交审核

## 注意事项

1. 网络请求域名需要在微信公众平台配置
2. 小程序包大小有限制，请注意优化资源文件
3. 需要后端API支持，可以使用原Next.js项目的API部分

## 技术特点

### 代码复用
- 配置数据完全复用原项目
- AI提示词模板逻辑一致
- 输入验证规则相同
- 业务逻辑保持不变

### 平台适配
- 使用微信小程序原生API替代浏览器API
- 采用WXSS替代Tailwind CSS
- 实现微信小程序组件系统
- 适配微信小程序生命周期
